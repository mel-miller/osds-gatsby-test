Home

This demo site is built with Gatsby to test the implementation of OSDS components. The stylesheet was added via the CDN and the markup was created via React Components.

Components

At this time the following components have example pages:

  • Accordions
  • Alerts
  • Buttons

Implementation

There are many ways to implement OSDS into a Gatsby project. This demo site uses the following methods:

CSS is implmented by importing the hosted CSS file into a global stylesheet for the project.

@import url("https://assets.osu.edu/osds/v0.x/css/osds-styles.min.css");

JS is implmented by importing the hosted JS file into the gatsby-ssr.js file for the project. This will place the script at the bottom of the <body> element.

import React from "react"
export const onRenderBody = ({ setPostBodyComponents }) => {
  setPostBodyComponents([
    <script src="https://assets.osu.edu/osds/v0.x/js/osds.min.js" />,
  ])
}
